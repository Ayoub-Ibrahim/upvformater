Help - UPV Markup to Moodle Format

Overview

Welcome to the UPV Markup Language to Moodle Converter help page. This web application is designed to simplify the process of converting question banks from the UPV Markup Language format into Moodle-compatible formats. It reduces the need for complex coding or spreadsheet manipulation, offering a straightforward and user-friendly interface for educators.

General Description

This web application allows educators to efficiently convert question banks from the UPV Markup Language format into Moodle-compatible formats. It streamlines the process, reducing the need for complex coding or spreadsheet use, and offers a simple, user-friendly interface for easy question conversion.

Getting Started

  1. Login or Create an Account: Before you can start converting your questions, ensure you are logged into the application. If you don’t have an account, create one using your email and a secure password.
  2. Accessing the Conversion Tool: Once logged in, navigate to the main dashboard where you can access the conversion tool for both multiple-choice and numeric questions.
  3. Inputting Questions: Use the provided text areas to input your questions using the UPV Markup Language syntax. You can also upload a text file containing multiple questions if preferred.

Tips for Efficient Use

  • Check Syntax: Before converting, ensure your questions are formatted correctly according to the UPV Markup Language syntax. This will prevent errors during the conversion process.
  • Use Bulk Upload: If you have multiple questions, consider using the bulk upload feature to save time. Prepare your questions in a single text file and upload them directly.
  • Preview Converted Questions: After conversion, preview your questions in Moodle format to verify accuracy before downloading or uploading them to your Moodle course.

Common Issues & Troubleshooting

  • Incorrect Formatting: If your converted questions don’t appear as expected, double-check the original UPV Markup Language formatting for errors or missing tags.
  • Conversion Errors: In case of an error during conversion, try refreshing the page or re-uploading your file. If the issue persists, contact support.
  • Account Issues: If you encounter any problems with your account, such as login difficulties or password reset issues, use the ‘Forgot Password’ feature or reach out to customer support for assistance.
