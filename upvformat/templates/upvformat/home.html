General Description of the Project:

Use this Web App to convert UPV Markup Language format into Moodle format, which is widely used in online learning environments. The application simplifies the creation of question banks, reducing the need for extensive coding or complex spreadsheet manipulation. With a user-friendly interface, it allows teachers to input questions directly or upload text files.

Multi choice
Multi-Choice Questions:

Input the Question: Enter your multi-choice question in the text area, using the UPV syntax.
Correct and Incorrect Answers: indicate which options are correct and which are incorrect.
Convert and Export: Click the convert button to generate the Moodle format, then download and upload the converted file to your Moodle course.
